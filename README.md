# Karlsen Coin

A GPU-friendly and ASIC-resistant BlockDAG implementation powering
decentralized AI inference on consumer hardware.

## Our Core Ecosystem

| Repository | Description | Status |
|------------|-------------|--------|
| [rusty-karlsen](...) | Karlsen node (KarlsenHashV2 PoW + GHOSTDAG) | Stable |
| [karlsenai-amd](...) | AMD ROCm worker for distributed LLM inference | Beta |
| [karlsenai-nvidia](...) | NVIDIA CUDA worker (candle-rs based) | Beta |
| [karlsenai-orchestrator](...) | VPS orchestrator + model catalogs | Beta |
| [karlsenai-frontend](...) | Web chat interface | Beta |
| [karlsen-mobile](...) | Flutter wallet (iOS / Android) | Released |
| [karlsen-explorer](...) | Block explorer + indexer | Released |

## Quick Start

- **Mine KLS:** see [karlsenminer releases](...)
- **Earn KLS by running an AI worker:**
  - AMD GPU → [download AMD Worker](.../karlsenai-amd/releases/latest)
  - NVIDIA GPU → [download NVIDIA Worker](.../karlsenai-nvidia/releases/latest)
- **Use AI services:** https://ai.karlsencoin.org
